# future-of-id-images

This repository houses all images used in the **Future of ID** app.

## Directory Structure

```
images/
├── avatars/        # User and profile avatar images
├── backgrounds/    # Background images and patterns
├── icons/          # App icons and UI icon assets
├── illustrations/  # Illustrative graphics and artwork
├── logos/          # Brand and partner logos
└── screenshots/    # App screenshots for documentation and store listings
```

## Usage

Place images in the appropriate subdirectory based on their purpose. When referencing images in the Future of ID app, use the raw GitHub URL format:

```
https://raw.githubusercontent.com/BradleyAPierce/future-of-id-images/main/images/<category>/<filename>
```

## Supported Formats

- **PNG** – recommended for icons, logos, and illustrations (lossless)
- **JPG/JPEG** – recommended for photographs and backgrounds
- **SVG** – recommended for scalable vector graphics
- **WebP** – recommended for optimized web delivery

## Contributing

1. Add your image to the correct category folder under `images/`.
2. Use descriptive, lowercase, hyphen-separated filenames (e.g., `login-background.png`).
3. Open a pull request with a brief description of the image(s) added.
